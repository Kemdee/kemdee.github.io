Welcome to our Website

This web application extracts and visualises various data-sets from current situations around the world. It enables users to access a range of data sources including, earthquakes, weather, population and COVID-19 stats visualised on maps, charts etc. The earthquake mapping page allows users to select different feeds to display on the map while the weather page facilitates the display of a five-day weather forecast for locations searched by the user. Users can also explore data on the population and COVID-19 stats for African countries.
